# Cataclysm: DDA - Ultimate Survival Wiki

A complete, interactive wiki website for Cataclysm: Dark Days Ahead, generated from the Ultimate Survival Guide.

## Features

- **9 Major Sections** covering every aspect of the game
- **Cross-linked Content** - Topics reference each other for easy navigation
- **Powerful Search** - Find information instantly across all sections
- **Mobile Responsive** - Works on desktop, tablet, and mobile
- **Dark Theme** - Easy on the eyes for long reading sessions
- **Print Support** - Print any page for offline reference

## Quick Start

Simply open `index.html` in your web browser:

```bash
# On Windows
cdda-wiki\index.html

# On Mac/Linux
open cdda-wiki/index.html
```

Or double-click the `index.html` file to open it in your default browser.

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl + K` | Open search |
| `Escape` | Close search |
| `?` | Show help (in-game) |

## Tips for Using This Wiki

1. **Use Search** - Press Ctrl+K to search the entire wiki instantly
2. **Follow Links** - Click on cross-references to explore related topics
3. **Check Tags** - Articles have difficulty/type tags for quick context
4. **Print Pages** - Use the print button to save pages as PDF
5. **Random Discovery** - Click the dice for random page exploration

## About the Content

This wiki is based on the "Ultimate Cataclysm: Dark Days Ahead Survival Guide" -
