THE COMPLETE CATACLYSM: DARK DAYS AHEAD GUIDE

For 0.H-RELEASE Herbert (Stable) - FINAL EDITION

INTRODUCTION

Welcome to the apocalypse. This guide contains everything you need to survive, thrive, and eventually master the chaos of Cataclysm: Dark Days Ahead.

How to Use This Guide

This manual is organized by progression phases:

  • Days 1-7: Immediate survival - don't die
  • Days 8-30: Establishment - secure base and resources
  • Days 30-90: Power progression - mutations, CBMs, advanced gear
  • Days 90+: Mastery - endgame content and optimization

Each module builds on the previous. Don't skip ahead until you've mastered current concepts.

Critical First Steps (Do These NOW)

Critical Warning

Most deaths happen in the first 3 days. Follow these steps religiously:

  1. Configure Safe Mode immediately (Esc → Options → Safe Mode) - Set to 5 tiles
  2. Enable Auto-Pulp (Options → Auto Features) - Prevents zombie revival
  3. Set up Auto-Pickup for medical supplies, water, and ammunition
  4. Learn the Peek command (x) - Check corners before moving
  5. Master the Drainpipe escape (climb up when surrounded)

AUTO-ACTIONS CONFIGURATION (Day 1 Priority)

Access via: Esc → 2. Options → Auto Features

Auto-Pickup (Configure IMMEDIATELY)

Location: Options → Auto Features → Auto Pickup

Recommended Settings:

  • Enable: Auto Pickup - ON
  • Enable: Auto Pickup Adjacent - ON (picks up from nearby tiles)

Custom Rules (Add These):

  • *matches (for lighting fires)
  • *bandage (medical)
  • *antibiotic (lifesaving)
  • *water (clean)
  • *aspirin (pain management)
Important

Use * as wildcard. Be careful not to auto-pickup corpses or rotten food.

Auto-Pulp (Day 1 Essential)

Location: Options → Auto Features → Auto Pulp or Butcher

Settings:

  • Set to "Pulp Adjacent Corpses"
  • Why: Prevents zombie revival, saves countless keystrokes

Auto-Forage (Mid-Game)

Location: Options → Auto Features → Auto Forage

Usage:

  • Great for wilderness survival
  • Disable when traveling (slows movement)
  • Only forages bushes/trees

Auto-Notes (Day 1 Essential)

Location: Options → Auto Notes

Settings:

  • Enable Map Extras
  • Why: Marks events/goodies on map you may have missed
  • Essential for exploration

Safe Mode Configuration (CRITICAL - Day 1)

Location: Options → Safe Mode

Recommended Settings:

  • Safe Mode - ON
  • Safe Mode Proximity Distance - 5 tiles (default 0 = max view)
  • Auto-Peek - OFF (reduces friction)

Ignore List:

  • Add regular zombies once you can handle them
  • Never ignore: shocker zombies, hulks, turrets
  • Press ' to temporarily ignore specific monster
Critical

Safe mode + auto-pulp prevents 80% of early deaths. Configure immediately.

ZONE MANAGER MASTERY (Y Key)

The Zone Manager is CDDA's most powerful automation tool. Master it.

Access: Y key

Essential Zone Types

1. Unsorted Loot Zone

  • Where you dump items
  • Required for sorting to work
  • Can cover entire buildings
  • Zones can overlap

2. Custom Storage Zones

Organized by category:

  • Food zone
  • Tools zone
  • Medical zone
  • Books zone
  • Clothing zone
  • Ammo zone

3. Vehicle Storage Zones (Experimental 8339+)

  • Can use Zone Manager with vehicle storage spaces
  • Set zone on vehicle tile
  • Sort items directly into vehicle
  • Cannot be multiple tiles - one zone per tile

4. Pulp Zone (Massive Time Saver)

  • Create zone over corpses
  • Use Zone Activities (O) → Pulp
  • Pulps all corpses in zone
  • Much faster than individual pulping

5. Construction Blueprint Zone

  • Define construction area
  • NPCs or player can build multiple tiles at once
  • Mass construction without individual placement
  • Requires materials and tools

6. Farming Zone

  • Automated farming operations
  • Planting, tilling, harvesting
  • Works with NPCs

7. Disassembly Zone

  • Auto-dismantle vehicles/terrain
  • Great for salvage operations
  • Set up near wreckage

Zone Activities (O Key)

After setting zones, press O for:

  • Sorting: Moves items from unsorted to custom zones
  • Pulping: All corpses in pulp zones
  • Construction: Mass building
  • Farming: Automated crop management
  • Disassembly: Salvage operations

Zone Overlap Strategies

  • Unsorted zone can cover entire building
  • Specific zones overlap with unsorted
  • Vehicle zones on individual tiles
  • Most efficient: Large unsorted + small specific zones

Hidden Zone Detection

  • Closest hidden zone's distance shown in UI
  • Helps find zones in large bases
  • Toggle visibility in zone manager

REALITY BUBBLE (Corrected)

What It Is

The "reality bubble" is the area around your character where the game processes events. Size: roughly 60x60 tiles centered on player.

Strategic Implications

  • Zombies outside bubble don't move
  • Items don't spoil outside bubble
  • Crops don't grow outside bubble
  • Use this to your advantage

Horde Movement

  • Hordes exist on overmap
  • They move between overmap tiles
  • Can "teleport" into reality bubble
  • Listen for approach sounds

MODULE 3: POWER PROGRESSION (Days 30-90)

MUTATION SYSTEM (Current 0.H Mechanics)

How Mutations Work

Mutations come from:

  • Mutagens: Consumable serums
  • Primer + Catalyst: Two-part system
  • Irradiation: Random, dangerous

Key Concepts:

  • Mutations belong to categories (Plant, Beast, Insect, etc.)
  • Each category has a "threshold"
  • Crossing threshold = powerful post-threshold mutations

Threshold Crossing (Updated)

To cross a threshold:

  1. Get many mutations in one category
  2. Consume that category's specific catalyst
  3. Requires high instability (see below)
Pro Tip

Threshold crossing is permanent. Choose your path wisely.

Instability Management

Instability = Risk of bad mutations

  • Higher instability = more mutations but worse ones
  • Reduces slowly over time
  • Purifier reduces mutations AND instability

Recommended Mutation Paths

Purifier Strategy

  • Removes random mutations
  • High instability = higher chance of removal
  • Save before using (backup saves)
  • Expensive to craft

I WISH I KNEW SOONER (By Mortality Rate)

1. Safe Mode + Auto-Pulp (Prevents 80% of Deaths)

Configure immediately:

  • Safe Mode: 5 tiles
  • Auto-Pulp: Adjacent corpses

These prevent most early deaths from unexpected hordes and zombie revival.

2. Drainpipe Escape

When surrounded:

  1. Find drainpipe on building
  2. e (examine) → Climb up
  3. Zombies can't follow

Lifesaver in emergencies.

3. Noise Discipline

  • Walk, don't run
  • Don't smash furniture (disassemble instead)
  • Avoid guns early game
  • Close doors quietly

4. Zone Manager

Press Y. Set up:

  • Unsorted Loot zone (base dump)
  • Custom storage zones
  • Press O to sort automatically

Massive time saver.

5. Auto-Pickup Configuration

Set rules for:

  • *bandage (medical)
  • *water (hydration)
  • *antibiotic (infection cure)

6. Peek Command (x)

Check corners before moving:

x → direction

Prevents walking into hordes.

7. Grab Break (G)

When grabbed by zombies:

G or Ctrl + direction

Escape and survive.

8. Water Before Food

Dehydration kills faster than starvation. Prioritize:

  1. Find toilet water (safe in emergency)
  2. Craft stone pot
  3. Boil all water before drinking
